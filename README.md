# SQL-Driven Business Intelligence Analysis

### 📊 Project Overview
Using SQL queries to extract deep business insights from over 11,000 retail transaction records.

### 🛠️ Tech Stack
* **Language:** Python
* **Library:** PandasQL (for running SQL queries on DataFrames)
* **Environment:** Google Colab

### 🔍 Key Queries Performed
* **Revenue Analysis:** Calculated total revenue generated per product.
* **Operational Health:** Identified top 5 products with the highest cancellation rates.
* **Customer Insights:** Filtered high-value transactions to identify VIP segments.

### 💡 Business Impact
This analysis helps the business identify loss-making products and focus on high-revenue sales channels discovered through automated SQL queries.
